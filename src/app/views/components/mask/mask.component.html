COMO FUNCIONA?

Para utilizar as máscaras deste componente externo, nós precisamos informar alguns parâmetros e atributos para cada caso, portanto é importante entender o que deve ser feito pois os mesmos podem ser misturados com base na necessidade.

Parâmetros/atributos mais comuns:
  • mask
  • specialCharacters: string[]
  • prefix: string
  • suffix: string
  • dropSpecialCharacters: boolean | string[]
  • showMaskTyped: boolean
  • allowNegativeNumbers: boolean
  • clearIfNotMatch: boolean
  • leadZeroDateTime: boolean

AVISO

É possível utilizar máscaras tanto em um elemento texto puro, por meio de pipes do Angular OU também em um input, mas é de extrema importância que o input seja do tipo "text", caso contrário a máscara não funcionará.

EXEMPLOS E CÓDIGOS
99/99/9999 0*.00 000.000.000-99 AAAA SSSS UUUU LLLL
0 Dígitos numéricos (de 0 até 9 números) 9 Dígitos numéricos (de 0 até 9 números), opcionais A Letras (Maiúsculas ou minúsculas) e dígitos numéricos S Apenas letras (Maiúsculas ou minúsculas) U Apenas letras Maiúsculas L Apenas letras minúsculas
